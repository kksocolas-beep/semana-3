Resumen de Diferencias Clave
En la Tradicional: Los datos son independientes de las funciones. Si pasas un diccionario con llaves incorrectas a mostrar_informacion(), el programa fallará en tiempo de ejecución.

En la POO: Los datos (nombre, especie, edad) y el comportamiento (mostrar_informacion()) viven juntos de forma segura dentro del objeto. No puedes crear una Mascota sin cumplir con la estructura que define su constructor (__init__).
